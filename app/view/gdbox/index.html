概述

特性

  • 多类型要素展示：图片/矢量/文本/标注
  • 高效率绘图：canvas矢量数据绘制

gDBox.Map

实例

// js: 伪代码
const gMap = new gDBox.Map(containerId, config);
// js: demo【gMap将作为后续使用的容器实例，不再进行重复实例操作】
const gMap = new gDBox.Map('map', {zoom: 640, cx: 0, cy: 0, zoomMax: 650 * 10, zoomMin: 650 / 10});

Params:

参数类型 是否必须默认 其他
containerIdstring 是
configConfig 是

Config:

参数类型 是否必须默认 其他
zoomnumber 是初始缩放级别
cxnumber 是初始横坐标
cynumber 是初始纵坐标
zoomMaxnumber 否无极缩放最大级别
zoomMinnumber 否无极缩放最小级别

事件

gDBox.Map支持各类事件监听。 
// js: 伪代码
gMap.events.on(eventType, callback);
// js: demo
gMap.events.on('mouseDown', xy => {console.log('xy');});

Params:

参数类型 是否必须类别 其他
eventTypestring 是见下
callbackfunction 否 mouseDown：wxy => {}
mouseMove：wxy => {}
geometryEditing：(type, feature, newPoints) => {}
geometryEditDone：(type, feature, newPoints) => {}
geometryDrawDone：(type, points) => {}
featureHover：feature => {}
featureSelected：feature => {}
featureStatusReset：() => {}

快捷键

键盘组合作用 应用模式其他
ctrl_z绘制过程中撤销 drawRect、drawPolygon
shift多边形绘制模式按下shift键进行选择操作 drawPolygon
shift多边形绘制模式按下shift键进行选中图形顶点delete操作 drawPolygon

addLayer

给gMap实例上添加图层。 
// js: 伪代码
const gFeatureLayer = new gDBox.Layer.Feature(layerId, config); // 请参考gDBox.Feature.Layer【此处不做赘述】
gMap.addLayer(gFeatureLayer); // 图层添加

removeLayer

给gMap实例上移除图层。 
// js: 伪代码
const gFeatureLayer = new gDBox.Layer.Feature(layerId, config); // 请参考gDBox.Feature.Layer【此处不做赘述】
gMap.addLayer(gFeatureLayer); // 图层添加
gMap.removeLayer(gFeatureLayer); // 图层移除

getAllLayers

获取gMap实例上所有图层。 
// js: 代码
gMap.getAllLayers(); // 返回[](layer)

getZoom

获取gMap实例当前缩放zoom值。 
// js: 代码
gMap.getZoom();

setZoom

设置gMap实例zoom值。 
// js: 代码
gMap.setZoom(zoom);

Params:

参数类型 是否必须默认 其他
zoomnumber 是

zoomIn

gMap实例放大。 
// js: 代码
gMap.zoomIn(num);

Params:

参数类型 是否必须默认 其他
numnumber 否6越大代表缩放的范围越大

zoomOut

gMap实例缩小。 
// js: 代码
gMap.zoomOut(num);

Params:

参数类型 是否必须默认 其他
numnumber 否6越大代表缩放的范围越大

setMode

gMap实例设置当前模式，同时可设置当前style样式。 
// js: 代码
gMap.setMode(mode[, gStyle]);

Params:

参数类型 是否必须默认 其他
modestring 是 'map': 浏览模式
'drawRect': 矩形绘制
'drawPolygon': 多边形绘制 		模式设置
gStylegDBox.Style 否参考gDBox.Style

getMode

获取gMap实例当前模式。 
// js: 代码
gMap.getMode();

setCenter

gMap实例设置中心点坐标。 
// js: 代码
gMap.setCenter(cx, cy);

Params:

参数类型 是否必须默认 其他
cxnumber 是横坐标
cynumber 是纵坐标

centerAndZoom

gMap实例设置中心点坐标并且缩放至指定值。 
// js: 代码
gMap.centerAndZoom(center, zoom);

Params:

参数类型 是否必须默认 其他
centerobject 是如：{x: 0, y: 0}
zoomnumber 是缩放值

getCenter

获取中心点。 
// js: 代码
gMap.getCenter();

getScreenCenter

获取屏幕中心点坐标。 
// js: 代码
gMap.getScreenCenter();

resize

gMap实例resize。 
// js: 代码
gMap.resize(w, h);

Params:

参数类型 是否必须默认 其他
wnumber 是
hnumber 是

destroy

gMap实例销毁。 
// js: 代码
gMap.destroy();

gDBox.Feature.Polygon

要素数据：多边形

实例

多边形要素实例。 
const featureLayer = new gDBox.Layer.Feature(layerId, config);
const feature = new gDBox.Feature.Polygon(featureId, points, data, gStyle);

gFeatureLayer.addFeature(feature);

Params:

参数类型 是否必须默认 其他
featureIdstring 是要素id
pointsarray 是要素空间数据，如[{x: y:}, {x: y:}, {x: y:}]，长度不能低于3个
dataobject 是要素属性数据，可通过实例.data属性获取
gStylegDBox.style 是要素样式

show

多边形要素显示。 
const featureLayer = new gDBox.Layer.Feature(layerId, config);
const feature = new gDBox.Feature.Polygon(featureId, points, data, gStyle);
gFeatureLayer.addFeature(feature);
feature.hide();

feature.show();

hide

多边形要素隐藏。 
const featureLayer = new gDBox.Layer.Feature(layerId, config);
const feature = new gDBox.Feature.Polygon(featureId, points, data, gStyle);
gFeatureLayer.addFeature(feature);

feature.hide();

update

多边形要素更新。 
const featureLayer = new gDBox.Layer.Feature(layerId, config);
const feature = new gDBox.Feature.Polygon(featureId, points, data, gStyle);
gFeatureLayer.addFeature(feature);

feature.update(options);

Params:

参数类型 是否必须默认 其他
optionsOptions 是待更新数据

Options:

参数类型 是否必须默认 其他
pointsarray 否待更新空间数据
dataobject 否待更新属性数据
stylegDBox.Style 否待更新样式数据

pointInFeature

判断点是否在当前要素内。 
const featureLayer = new gDBox.Layer.Feature(layerId, config);
const feature = new gDBox.Feature.Polygon(featureId, points, data, gStyle);
gFeatureLayer.addFeature(feature);

feature.pointInFeature(wxy); // return false;

Params:

参数类型 是否必须默认 其他
wxyobject 是如{x: , y: }
