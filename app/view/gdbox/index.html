概述

在前端开发过程中难免遇到针对图片的缩放平移；以及在图片上进行矢量数据、文本、标注的展示； 如果你有上面的任何需求，恭喜你，找到组织了....在此背景下，gDBox.js诞生了

特性

  • 多类型要素展示：图片/矢量/文本/标注
  • 高效率绘图：canvas矢量数据绘制
  • ...

gDBox.Map

实例

// js: 伪代码
const gMap = new gDBox.Map(containerId, options);
// js: demo
const gMap = new gDBox.Map('map', {zoom: 640, cx: 0, cy: 0, zoomMax: 650 * 10, zoomMin: 650 / 10});

params:

参数类型 是否必须默认 其他
containerIdstring 是
optionsobject 是
options.zoomnumber 是初始缩放级别
options.cxnumber 是初始横坐标
options.cynumber 是初始纵坐标
options.zoomMaxnumber 否无极缩放最大级别
options.zoomMinnumber 否无极缩放最小级别
